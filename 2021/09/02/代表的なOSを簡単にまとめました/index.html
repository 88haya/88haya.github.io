代表的なOSを簡単にまとめました

OSとはオペレーティング・システムの略で基本ソフトウェアとも訳され,コンピュータの基本的な動作や機能を担当するソフトウェアです.

パソコンではWindowsやMacOSが有名で,スマートフォンではiOSやAndroidOSが有名です.

この記事ではパソコンのOSとして普及した代表的なものについて簡単にまとめます.

GUI/CUIとは？

後の説明のためにまずGUIとCUIについて解説します.

コンピュータを操作する方法にはGUIとCUIという２種類があります.

GUIとは

GUIとはGraphical User Interfaceの略で,コンピュータの操作を主にマウスを用いて行う方法です.

現在多くの人がパソコンを操作するときに用いている方法はGUIです.

CUIとは

CUIとはCharacter User Interfaceの略で,characterには「文字」という意味があります.
コンピュータの操作を全てキーボードを用いて行う方法です.

ドラマや映画などでハッカーやコンピュータの達人のような人が真っ黒の画面に白い文字で色々と入力しているのをよく見ると思いますがあれはCUIでコンピュータを操作しています.

一般の人はほとんど利用することはありませんが,ITエンジニアや情報工学を専攻している学生はよく利用します.
WindowsやMacOS,Linuxでも当然利用できます.

MS-DOS

MSとはmicrosoft,DOSとはDisk Operating Systemの略で,Windowsが普及する前に広く利用されていたMicrosoft社製のOSです.

CUIでしか操作できません.
現在はほとんど利用されていません.

Windows

現在最も普及しているMicrosoft社製のOSです.
Window95,Windows7,Windows8,Windows10などのバージョンが有名です.

GUIでの操作が基本です.

MacOS

Apple社製のOSです.
GUIで動作するOSの先駆けです.(Windowsがパクったという説もあります.)

実態はUNIX(後述)です.

Windowsはどのパソコン(東芝のdynabookでもLenovoでもDellでもマウスコンピュータでも)動作しますが,MacOSはApple製のパソコン(MacbookやiMacなど)でしか動作しません.

UNIX

現存する最も古いOSです.

サーバなどでよく用いられます.

Linux

UNIXを参考にして作られて今でもITエンジニアの間では人気の高いOSです.

オープンソース(ソースコードが無償で公開されている)であり,無償で利用できる.
つまり元々WindowsやMacOSが入っているパソコンを買って後からLinuxをインストールしてLinuxとして利用することもできる.
