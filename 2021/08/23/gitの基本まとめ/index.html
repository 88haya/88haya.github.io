Gitとは/GitHubとは

Gitとは,複数人で共同開発を行うときに主に用いられ,ソースコードなど開発に必要なファイル等を共有するシステムです.

GitHubとは,クラウド上でGitを管理するWebサービスです.
GitHub

GitとGitHubの関係はメールとGmailの関係に似ています.
